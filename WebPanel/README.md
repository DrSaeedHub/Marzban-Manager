# Marzban Manager Web Panel

## Project info

This repository contains the web panel for Marzban Manager.

## Development

Requirements: Node.js and npm.

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading.
npm run dev
```

## Editing options

- Use your preferred IDE locally.
- Edit directly in GitHub.
- Use GitHub Codespaces.

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Build the production assets with `npm run build` and deploy the generated `dist` folder to your hosting provider.
